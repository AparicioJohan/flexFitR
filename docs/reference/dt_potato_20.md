# Drone Data in Potato 2020

Vegetative indices and biomass related traits for a potato trial
arranged in a p-rep design.

## Usage

``` r
dt_potato_20
```

## Format

A tibble with 1372 rows and 17 variables:

- Trial:

  chr trial name

- DAP:

  dbl denoting Days after planting

- Plot:

  dbl denoting the unique plot id

- Row:

  dbl denoting the row coordinate

- Range:

  dbl denoting range coordinate

- gid:

  chr denoting the genotype id

- Canopy:

  dbl Canopy UAV-Derived

- GLI:

  dbl Green Leaf Index UAV-Derived
