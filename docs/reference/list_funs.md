# Print available functions in flexFitR

Print available functions in flexFitR

## Usage

``` r
list_funs()
```

## Value

A vector with available functions

## Examples

``` r
library(flexFitR)
list_funs()
#>  [1] "fn_lin"        "fn_quad"       "fn_logistic"   "fn_lin_plat"  
#>  [5] "fn_lin_logis"  "fn_quad_plat"  "fn_quad_pl_sm" "fn_lin_pl_lin"
#>  [9] "fn_lpl"        "fn_lll"        "fn_qpl"        "fn_exp_exp"   
#> [13] "fn_exp_lin"    "fn_exp2_exp"   "fn_exp2_lin"  
```
